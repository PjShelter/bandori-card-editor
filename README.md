# Bandori Card Editor

🎸 自定义 Bandori 卡面布局的在线编辑工具

## Features

- 📤 上传自定义人物图片
- 🎛️ 调整图片位置与缩放
- 🎸 支持 9 大乐队图标
- 💎 4 种属性选择
- 🖼️ 多种卡框样式（2-5 星）
- ⭐ 特训前后状态切换
- 📥 导出 PNG 格式

## Usage

1. 上传人物透明背景图
2. 使用滑块调整位置和缩放
3. 选择乐队、属性、边框等
4. 点击导出按钮下载
